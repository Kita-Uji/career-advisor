# Roadmap: Career Advisor UI Polish

## Overview

The app is feature-complete. This single-phase project applies targeted visual polish to two component files — MessageBubble and ChatInput — so the chat interface feels calm, minimal, and trustworthy. All requirements deliver in one coherent change set.

## Phases

**Phase Numbering:**
- Integer phases (1, 2, 3): Planned milestone work
- Decimal phases (2.1, 2.2): Urgent insertions (marked with INSERTED)

Decimal phases appear between their surrounding integers in numeric order.

- [ ] **Phase 1: UI Polish** - Remove avatar clutter and elevate the input card for a cleaner, more professional feel

## Phase Details

### Phase 1: UI Polish
**Goal**: The chat interface looks calm and minimal — assistant bubbles are clean without an avatar, and the input feels like the focal point of the page
**Depends on**: Nothing (first phase)
**Requirements**: BUBBLE-01, BUBBLE-02, BUBBLE-03, INPUT-01, INPUT-02
**Success Criteria** (what must be TRUE):
  1. Assistant message bubbles display no avatar circle or CA initials
  2. Assistant message bubbles show a soft indigo-50 background instead of white
  3. User message bubbles remain visually unchanged (indigo-600 background)
  4. The chat input displays as a floating elevated card with shadow-lg and rounded-2xl, lifted off the bottom bar
